# Imago 开发计划 (极简图片处理工具)

本项目旨在构建一个 100% 浏览器端本地处理的图片工具站，确保隐私安全与极简体验。

## 技术栈
- **框架:** Vue 3 (Composition API) + Vite
- **状态管理:** Pinia
- **样式:** 原生 CSS / CSS 变量 (遵循轻量级原则)
- **核心库:**
- `browser-image-compression`: 图片压缩
- `exifr`: EXIF 数据处理
- `jszip`: 批量打包下载
- `lucide-vue-next`: 图标库

## 核心功能路线图

### 第一阶段：基础框架与上传系统 (完成)
- [x] 项目初始化 (Vite + Vue 3 + TS)
- [x] 通用布局 (侧边栏导航 + 中央处理区)
- [x] 智能上传组件 (支持拖拽、粘贴、点击上传)
- [x] 基础 Composable: `useImageProcessor` (正在集成)

### 第二阶段：基础工具集 (进行中)
- [x] **图片压缩 (Compress):** 质量控制、尺寸限制、实时预览对比。
- [ ] **调整尺寸 (Resize):** 按比例、指定宽高、批量处理。
- [ ] **格式转换 (Convert):** JPG, PNG, WebP, AVIF 互转。
- [ ] **基础裁剪 (Crop):** 自由裁剪、1:1, 16:9, 圆形裁剪。

### 第三阶段：进阶工具集
- [ ] **图片分割 (Split):** 网格分割、行列分割。
- [ ] **隐私清理 (EXIF):** 查看并擦除元数据。
- [ ] **滤镜增强 (Filters):** 亮度、对比度、灰度、反色、像素化。
- [ ] **图片合并 (Combine):** 横向/纵向拼接。

### 第四阶段：体验优化
- [ ] PWA 支持 (离线使用)
- [ ] 批量任务排队处理
- [ ] 多语言支持
- [ ] 暗黑模式切换

## 设计原则
1. **隐私优先:** 绝不上传文件到服务器。
2. **极简主义:** 无需注册，即开即用。
3. **响应式:** 完美适配移动端和桌面端。
